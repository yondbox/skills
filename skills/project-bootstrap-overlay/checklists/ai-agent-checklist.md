# AI Agent Checklist

## Instruction Files

- [ ] AGENTS.md exists or is intentionally omitted
- [ ] CLAUDE.md exists only when useful
- [ ] .github/copilot-instructions.md exists only when useful
- [ ] GEMINI.md exists only when Gemini CLI is used

## Skills / Hooks / Subagents

- [ ] skills exist only for repeated procedures

- [ ] subagents have clear roles

## Quality

- [ ] AGENTS.md is short
- [ ] verification expectations are clear
- [ ] dependency rules are clear
- [ ] AI is instructed not to touch secrets
